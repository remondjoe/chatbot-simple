Panduan API Key

Bantuan

Untuk menggunakan SiCerdas dengan akun sendiri, Anda memerlukan Google Gemini API Key. Gratis kok! Ikuti langkah ini:

  1. Kunjungi Google AI Studio
    Buka link ini: https://aistudio.google.com/app/apikey
  2. Login dengan Akun Google
    Masuk menggunakan akun Gmail Anda.
  3. Buat Kunci API
    Klik tombol besar bertuliskan "Create API key".
  4. Pilih Proyek
    Jika diminta, pilih "Create API key in new project".
  5. Salin Kunci Anda
    Salin kode panjang yang muncul (dimulai dengan AIza...).
  6. Gunakan di SiCerdas
    Kembali ke halaman Daftar dan tempelkan kodenya di kolom "API Key".
